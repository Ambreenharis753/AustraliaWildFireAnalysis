# 🔥 Australia Wildfire Analysis & Interactive Dashboard
## 📌 Project Overview

This project analyzes wildfire activity across multiple regions of Australia from 2005 to the present using data analytics, geospatial visualization, and interactive dashboards.

The goal of the project was to explore spatiotemporal wildfire patterns, identify high-risk regions, and develop interactive visualizations to support environmental analysis and decision-making.

The project combines:

exploratory data analysis (EDA)

statistical visualization

geospatial mapping

interactive dashboard development

to better understand wildfire behavior across Australia.

## 🎯 Project Objectives

The main objectives of this project were to:

Analyze wildfire trends across Australian regions

Identify seasonal and regional wildfire patterns

Explore relationships between fire area, brightness, radiative power, and confidence levels

Build interactive dashboards for dynamic data exploration

Develop geospatial visualizations to highlight wildfire severity zones

# 📈 Key Analytical Insights

## 🔥 Fire Brightness Trends

Using Seaborn and Matplotlib, the analysis revealed that:

Mean estimated fire brightness increased by approximately 30% in high-risk regions during peak wildfire months.

## 🌍 Regional Severity Patterns
